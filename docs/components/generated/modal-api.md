| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| show | `boolean` | false | 是否显示弹窗 |
| title | `string` | '' | 弹窗标题 |
| width | `number \| string` | 520 | 弹窗宽度 |
| preset | `'dialog' \| 'card'` | 'card' | 弹窗预设样式 |
| maskClosable | `boolean` | true | 点击遮罩是否关闭弹窗 |
| closable | `boolean` | true | 是否显示关闭按钮 |
| bordered | `boolean` | false | 是否使用内置边框 |
| autoFocus | `boolean` | true | 是否自动聚焦弹窗内容 |
